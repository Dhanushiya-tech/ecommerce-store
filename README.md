# 🛒 E-commerce Store (Full Stack)

This is a full-stack e-commerce store application built with:

- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Features**:
  - Product listing
  - Shopping cart
  - User login/signup
  - Checkout page
